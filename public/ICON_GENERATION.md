# PWA Icon Generation Guide

## Quick Start

To generate the required PWA icons:

1. **Open the icon generator tool:**
   - Navigate to `/generate-icons.html` in your browser
   - Or open `public/generate-icons.html` directly

2. **Generate and download icons:**
   - Click "Generate" buttons to preview icons
   - Click "Download" buttons to save each icon
   - Save them in the `/public` directory

3. **Required icons:**
   - `pwa-192x192.png` (192x192 pixels)
   - `pwa-512x512.png` (512x512 pixels)
   - `apple-touch-icon.png` (180x180 pixels) - Optional but recommended

## Custom Icons

For production, replace the placeholder icons with custom designs:

- **Background colors:** Use #3E2723 (leather) or #F5F5DC (canvas)
- **Icon:** Gold ⚜️ symbol or your brand logo
- **Format:** PNG with transparency support
- **Design tools:** Figma, Canva, Adobe Illustrator, or online PWA icon generators

## Online Tools

- [PWA Builder Image Generator](https://www.pwabuilder.com/imageGenerator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [Favicon.io](https://favicon.io/)

## Notes

- Icons are required for the PWA to be installable
- The placeholder icons generated by the HTML tool will work for testing
- Replace with custom branded icons before production deployment
